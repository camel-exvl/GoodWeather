# Good Weather

## Config

### 和风天气API配置

在`/app/src/main/res/values`下添加文件`secret.xml`，内容如下：

```xml
<?xml version="1.0" encoding="utf-8"?>
<resources>
	<string name="qweather_key">your_key</string>
</resources>
```
